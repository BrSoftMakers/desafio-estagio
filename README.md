# desafio-estagio
---
### como excecutar o projeto localmente

## Para o backend

Instalar:

```
yarn install 
```

Executar

```
yarn start
```

## Para o frontend

Entre na pasta do frontend

```
cd frontend
```

Para instalar 

```
yarn install 
```

Para executar

```
yarn serve
```

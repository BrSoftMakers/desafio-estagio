# Rodando localmente

Clone o projeto

  ```
  git clone git@github.com:larissaassilva/desafio-estagio.git
  ```
  
  Entre no diretório do projeto
  
  
  ```
  cd desafio-estagio
  ```

  Instale as dependências

  ```
  npm install
  ```

  Para ver a aplicação no navegador 

  ```
  npx vite --open
  ```
